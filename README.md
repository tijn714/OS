# OS
start of my OS development journey.

# ChangeLog:
- v1 (Current): Basic kernel that does nothing, startpoint of the learning progress. 

# Requirements:
- i686-elf-toolchains
- grub dev toolchain
- make
- QEMU (to run the ISO image)

# make the ISO using:
```
make iso
```
- to build the ISO
```
make run
```
- to emulate the ISO with QEMU
